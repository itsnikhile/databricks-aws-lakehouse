🗄 Layer 2 — AWS S3 Data Lake (Delta Lake Format)
🥉 Bronze Zone (Raw)
Raw, unprocessed data. Immutable append-only. Partitioned by ingestion date. Parquet + JSON. Schema-on-read.
🥈 Silver Zone (Cleaned)
Validated, deduplicated, type-cast data. MERGE operations. PII masked. Delta format with ACID transactions.
🥇 Gold Zone (Curated)
Business-level aggregates, star/snowflake schemas, feature stores, ML-ready datasets. Optimized for query.
🔵
Delta Lake
ACID, time travel, schema enforcement & evolution, Z-order clustering
Databricks
🪣
